Safekeeping public COVID-19 serological data for reference. Click here to download data as JSON or as TSV (tabular). The data originally at covid19serohub.nih.gov, is now maintained here in GitHub, for long-term versioned availability. A number of interactive JavaScript visualizations and applications will be added to this page over time. If you want to suggest new ones, or to report problems with the existing tools, please use the issues link above. Note also that an SDK is available for programatic operation on the reference dataset (see Notebook above). Finally, for information and attribution describing this resource, please consult and contribute (via Issues) to the Documentation Wiki, above.
